
## Electric blanket blueprint

### Installation
Before installing the blueprint, create helpers similar to the following in Settings → Devices & services → Helpers:<p>
switch.power: The switch controlling the blanket<br>
input_button.start_button: The button to start and increment the blanket timer<br>
input_button.clear_button: The button to stop the blanket timer<br>
timer.blanket_timer: The timer to control the blanket<br>
number.timer_inc: Timer increment step in seconds<br>
number.timer_max_inc: Maximum timer of increments permitted, to limit on time<br>
